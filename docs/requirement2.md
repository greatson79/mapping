제목:
대규모 기능 추가 및 수정 계획

목표:
기존에 비로그인으로, 단순 텍스트 리뷰만 남길 수 있었으나 다음 기능들을 추가하여 더 풍부한 경험을 제공한다.

- 하단 탭바 추가 (홈 / 탐색 / 마이페이지)
- 비로그인 상태로 홈/탐색 접근 시 중앙에 로그인 요구 modal 표시. 버튼 클릭해 로그인 페이지로 이동
- 카카오 연동 로그인/회원가입 추가. 닉네임 추가 입력 가능.

페이지별 요구사항:

- 홈: '/', 메인페이지, 기존 작동을 유지
- 탐색: 모든 장소 목록이 표시됨. 별점순, 최근 리뷰순, 리뷰많은순 등 정렬 가능
- 마이페이지: 사용자 정보, 리뷰 내역, 통계 확인 가능. 로그아웃 버튼 존재.
