## 만들 것

- '네이버 지도SDK,API를 연동한 맛집리뷰 플랫폼'

- 네이버 지도 SDK, api를 사용한다.
  - 장소 검색 및 리뷰 작성
  - 지도 표시
  - 마커 표시
- 비회원으로만 사용한다.
- 모든 페이지는 모바일로 최적화되며, 모두 모바일 모양처럼 최대 너비 360 / 최대 높이 720을 갖는 MainLayout 내부에 렌더링된다.

## 주의 사항

- 반드시 비회원으로만 사용한다.

## 페이지 목록

- 홈 ('/', 메인페이지, 네이버 지도 SDK로 기본 지도 렌더링.)
  - 보여지고 있는 지도 영역에 리뷰 데이터가 존재하는 장소들에 대해, 적절히 마커 표시.
  - 마커를 누르면 해당 장소의 '장소 상세' 페이지로 이동한다.
  - 상단에 장소 검색용 입력창 존재, 입력창 우측에 search button 표시해 제출 가능.
  - 제출 시 입력한 키워드로 네이버 장소검색 API 호출한 뒤, 응답이 오면 장소 목록 dialog 표시
  - 장속 목록 dialog 각 항목의 '리뷰 작성' 버튼을 눌러 '리뷰 작성' 페이지로 이동할 수 있다.
- 리뷰 작성 ('/review/create')
  - 선택한 장소 정보가 보인다. 네이버 장소검색 API에서 반환된 데이터로 적절히 구성한다.
  - 작성자명, 평점, 본문, 4자리 이상의 임시비밀번호를 입력 후 제출한다.
  - 데이터 저장이 완료되면 적절히 완료상태를 보여준 뒤, 확인 버튼을 눌러 메인페이지로 돌아갈 수 있다.
- 장소 상세 ('/place/detail')
  - 선택한 장소 정보가 보인다. 네이버 장소검색 API에서 반환된 데이터로 적절히 구성한다.
  - 해당 장소의 리뷰 개수, 평균 평점이 보인다.
  - 해당 장소에 작성된 리뷰 목록이 보인다. 리뷰 아이템을 클릭해 '리뷰 상세' 페이지로 이동할 수 있다.
